# pv-power-monitoring-system

An embedded systems project developing a real-time power monitoring 
system for a photovoltaic (PV) panel.

## Overview
This project involved designing and building a functional PV power 
monitoring system that reads live power output from a solar panel 
and displays it in real time on an LCD screen, using an Arduino 
microcontroller and custom power sensing circuitry.

## Tools & Technologies
- Arduino Uno R3: microcontroller platform
- C++: embedded firmware programming
- Power sensing circuitry: designed circuit with INA219 current sensor module
- LCD display: real-time output visualisation

## What I Did
- Designed and developed power sensing circuitry to measure PV 
  panel electrical output
- Programmed and validated embedded firmware in C++ to interface 
  with peripheral hardware modules and process real-time electrical 
  measurements
- Implemented LCD display output to visualise live power readings
- Demonstrated embedded systems development from 
  circuit design through to firmware implementation

## Key Skills Demonstrated
- Embedded systems programming
- Microcontroller interfacing and hardware integration
- C++ firmware development
- Sensor integration and signal processing
- Hardware-software co-design

## Results
Delivered a fully functional PV monitoring system capable of reading 
and displaying real-time power output, demonstrating practical 
embedded systems and electrical design skills.
